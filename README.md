# To-Do List

Uma aplicação web simples para gerenciar listas de tarefas, desenvolvida com HTML, CSS e JavaScript.

---

## 📸 Preview

![Preview do Projeto](/assets/preview.png)

---

## 📋 Funcionalidades

- Adicionar novas tarefas
- Marcar/desmarcar tarefas como concluídas (clicando no texto)
- Remover tarefas individuais
- Remover todas as tarefas concluídas de uma vez
- Persistência automática de dados (localStorage) - as tarefas permanecem salvas mesmo após fechar o navegador
